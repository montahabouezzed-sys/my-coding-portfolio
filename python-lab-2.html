Advanced Data Processing

In this project, I used Python to filter a list of numbers, keeping only the ones that meet specific criteria. This is a fundamental concept in data science and backend logic.


# A script to filter high-value transactions
transactions = [10.50, 100.00, 5.25, 500.0, 12.00]
    high_value = []
            
    def filter_data(data_list):
        for item in data_list:
            if item > 50:
                high_value.append(item)
            return high_value
    
        print(f"High value items: {filter_data(transactions)}")
What I learned:
  • Iterating through lists using for loops.
  • Conditional filtering with if statements.
  • The difference between local and global variables.

Efficient Data Grouping

I used a dictionary to categorize flat data into logical groups. This reduces the complexity of searching through the data and prepares it for statistical analysis (like calculating mean temperature).


# Raw data: A list of dictionaries (common in API results)
sensor_data = [
    {"id": "SN-001", "type": "Temp", "value": 22.5},
    {"id": "SN-002", "type": "Vibration", "value": 0.5},
    {"id": "SN-003", "type": "Temp", "value": 23.0},
    {"id": "SN-004", "type": "Pressure", "value": 101.3}
]

# Dictionary Manipulation: Grouping values by their type
analysis = {}

for record in sensor_data:
    s_type = record["type"]
    val = record["value"]

    if s_type not in analysis:
        analysis[s_type] = []  # Initialize a new list for this type

    analysis[s_type].append(val)

print(f"Mapped Data: {analysis}")
# Output: {'Temp': [22.5, 23.0], 'Vibration': [0.5], 'Pressure': [101.3]}
What I learned:
  • Structuring information with "keys" and "value through lists using for loops.
  • Categorizing flat data into logical groups for easier processing and calculation afterwards.
